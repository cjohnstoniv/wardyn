## AI Run Composer (new feature)

`POST /api/v1/runs/compose` + `GET /api/v1/composer/backends` (opt-in via
`-composer-config`; 404 when disabled). Turns a natural-language task into a
PROPOSED `{run, inline_policy}` for human approval. Multi-provider behind one
`Composer` interface: **Anthropic** (API key, Bedrock), **OpenAI** (API key, Azure
Foundry, any OpenAI-compatible BYOK/BYOM), **subscriptions** (claude/codex CLI,
opt-in). Security (OWASP LLM01): advisory-only; risk graded **deterministically by
Wardyn** from the spec (never the model's self-assessment); **clamped to operator
policy**; source URLs never fetched; untrusted attachments fenced; the adversarial
test proves a prompt-injected attachment can neither raise privileges nor lower
its grade.
